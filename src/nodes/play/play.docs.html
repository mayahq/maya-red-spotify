Description

Play a song, album, playlist, or artist

Inputs

uri string
The URI of the spotify resource you want to play. For example, spotify:track:4iV5W9uYEdYUVa79Axb7Rh is a URI for a track. Songs, artists, albums and playlists all have unique URIs associated with them. You can read more about URIs here

Outputs

This node does not have any outputs. If the play operation is successful, it simply forwards the message.
